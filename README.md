## Лабораторные работы

### Лаб. 1 — Вычислительная погрешность
> `task_1.ipynb`

Вычисление суммы $S = \sum_{n=1}^{10^6} \frac{\pi}{n^2}$ двумя способами:

- **Прямое суммирование** ($1 \to N$)
- **Обратное суммирование** ($N \to 1$)

Сравнение абсолютной и относительной погрешности. Разбор причин потери точности в арифметике с плавающей точкой (поглощение мантиссы, неассоциативность сложения в `float`).

### Лаб. 3 — Решение СЛАУ методом LU-разложения
> `task_3.ipynb`

Решение системы $Ax = b$ методом LU-разложения (по лекции 2: «Прямые методы решения СЛАУ»).

**Алгоритм (3 этапа):**

1. LU-разложение: $A = LU$, где $l_{jk} = \dfrac{a_{jk}}{a_{kk}}$
2. Прямая подстановка: $Ly = b \to y$
3. Обратная подстановка: $Ux = y \to x$

Реализация на чистом Python (циклы + условные операторы, без `numpy` в ядре алгоритма). Проверка: $L \cdot U = A$, невязка $\|Ax - b\| = 0$, сверка с `numpy.linalg.solve`.
